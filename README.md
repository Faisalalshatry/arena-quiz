# ARENA — منصة المسابقات التفاعلية

منصة مسابقات تفاعلية تدعم أكثر من فريق، مع تزامن فوري عبر Firebase Realtime Database.

## المميزات
- 👥 فرق غير محدودة مع أعضاء وأوصاف
- ❓ ثلاثة أنواع من الأسئلة: اختيار متعدد، كتابي، زر سريع
- ⚡ تزامن فوري بين جميع الأجهزة
- ☀️ وضع نهاري وليلي
- 🔐 حماية لوحة المشرف برمز PIN
